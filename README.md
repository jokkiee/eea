# موقع نقابة المهندسين بالبحيرة | دليل الاستضافة والرفع

هذا المجلد يحتوي على كافة ملفات الموقع جاهزة للرفع والنشر المباشر على منصات الاستضافة المجانية الاحترافية مثل **Cloudflare Pages** أو **GitHub Pages** أو **Vercel** أو **Netlify**.

---

## 🚀 1. الرفع على Cloudflare Pages (الموصى به - مجاني وسريع جداً)

1. سجل الدخول إلى حسابك في [Cloudflare Dashboard](https://dash.cloudflare.com/).
2. انتقل إلى قائمة **Workers & Pages** -> اختر **Create Application** -> ثم **Pages**.
3. قم بتوصيل حساب **GitHub** واختر المستودع (Repository)، أو اختر **Direct Upload** وارفع مجلد `test website` مباشرة بمجرد سحبه وإفلاته (Drag & Drop).
4. اضغط على **Save and Deploy**.
5. ستحصل فوراً على رابط مجاني ينتهي بـ `.pages.dev` مجهّز بشهادة أمان SSL تلقائياً.

---

## 🐙 2. الرفع على GitHub Pages

1. قم بأنشاء مستودع جديد (Repository) على حسابك في GitHub باسم `behira-engineers-website`.
2. ارفع كافة الملفات الموجودة داخل مجلد `test website` إلى المستودع.
3. الذهاب إلى **Settings** الخاصة بالمستودع -> اختر **Pages** من القائمة الجانبية.
4. في قسم **Branch** اختر `main` أو `master` ثم اضغط **Save**.
5. خلال ثوانٍ سيكون موقعك متاحاً على الرابط: `https://username.github.io/behira-engineers-website/`
